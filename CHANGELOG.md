# 更新日志 (Change Log)

本项目的所有重要更新都将记录在此文件中。

本文件遵循 [Keep a Changelog](http://keepachangelog.com/) 规范。

## [1.0.2] - 2026-01-28
### ✨ 新增 (Added)
- **快捷交互菜单**：点击状态栏图标现可唤出快捷菜单，无需打开命令面板即可快速操作。
- **一键开关**：新增 `codeStatus.toggleEnabled` 命令，支持在不重启插件的情况下随时开启或暂停同步。

### ⚡ 优化 (Changed)
- 优化了状态栏的交互体验，从单一按钮改为多功能菜单。
- 重构了命令注册逻辑，提升了插件响应性能。
- 默认游戏从PEAK 修改至 CS2
- `README.md` 增加致谢
- bridge的steam群组和群组人数值改为缺省，可以不传

## [1.0.1] - 2026-01-27
### 🐛 修复 (Fixed)
- 修复了发布到市场时安装包内缺失 `README.md` 说明文件的问题。
- 更新了文档，补充了关于 C# 后端桥接程序的编译指南。

## [1.0.0] - 2026-01-27
### 🎉 首次发布 (Initial Release)
- **核心功能**：实现 VS Code 编辑状态与 Steam 好友列表的实时同步。
- **自定义模板**：支持强大的模板语法（如 `[{projectName} | ]{fileName}`），并支持条件隐藏功能。
- **手动锁定**：支持通过命令强制锁定状态文本（Manual Override）。
- **稳定架构**：引入独立的 C# Bridge 后端进程，确保与 Steamworks API 的通信稳定可靠。
